# ClipBoard - macOS 历史剪切板工具

![macOS](https://img.shields.io/badge/macOS-13.0+-orange)
![Swift](https://img.shields.io/badge/Swift-5.9-blue)
![License](https://img.shields.io/badge/License-MIT-green)

一个简洁高效的 macOS 菜单栏剪切板历史管理工具，帮助你管理复制历史，提升工作效率。

---

## ✨ 功能特点

### 核心功能
- **全局快捷键** - `⌘⇧V` 快速唤起/隐藏面板
- **剪切板监控** - 自动记录文本、图片、文件路径
- **历史浏览** - 卡片式横向滚动展示，清晰直观
- **一键复制** - 点击历史项自动复制并粘贴
- **多类型支持** - 文本、图片、文件路径均可记录

### 用户体验
- **菜单栏应用** - 无 Dock 图标，不打扰工作
- **底部弹出** - 面板从屏幕底部滑出，不遮挡内容
- **点击外部关闭** - 点击面板外部区域自动隐藏
- **深浅色适配** - 自动跟随系统外观模式
- **流畅动效** - 面板显示/隐藏带有柔和动画

### 设置选项
- **保留天数** - 可配置 7/30/90/180 天
- **开机自启** - 可选择开机自动启动

## 🚀 安装与运行

### 方式一：下载安装包
1. 下载最新版本的 `.dmg` 安装包
2. 打开安装包，将应用拖入 `Applications` 文件夹
3. 首次运行需要在「系统设置 → 隐私与安全性」中授权

### 方式二：从源码编译
```bash
# 克隆仓库
git clone https://github.com/LGN1995/ClipBoard.git

# 进入项目目录
cd ClipBoard

# 使用 XcodeGen 生成项目（需要先安装 XcodeGen）
xcodegen generate

# 用 Xcode 打开并编译运行
open ClipBoard.xcodeproj
```

### 环境要求
- macOS 13.0 (Ventura) 或更高版本
- Xcode 15.0+ (如需源码编译)

## ⌨️ 使用说明

### 唤起面板
| 操作 | 快捷键 |
|------|--------|
| 显示/隐藏面板 | `⌘⇧V` |
| 点击菜单栏图标 | - |

### 快捷操作
| 操作 | 说明 |
|------|------|
| 点击历史卡片 | 复制内容并自动粘贴 |
| 鼠标悬停卡片 | 显示 Copy 提示 |
| 点击面板外部 | 隐藏面板 |

### 菜单栏选项
- **显示面板** - 唤起历史面板
- **设置...** - 配置保留天数和开机自启
- **清空历史** - 删除所有历史记录
- **退出** - 退出应用

## 🏗 技术架构

### 技术栈
| 分类 | 技术 |
|------|------|
| 语言 | Swift 5.9 |
| UI 框架 | SwiftUI + AppKit |
| 全局快捷键 | HotKey |
| 数据存储 | UserDefaults |
| 项目管理 | XcodeGen |

### 架构模式
```
ClipBoard/
├── App/
│   ├── main.swift              # 应用入口
│   ├── AppDelegate.swift       # 菜单栏 + 快捷键 + 面板管理
│   ├── Info.plist              # 应用配置（LSUIElement 等）
│   └── ClipBoard.entitlements # 权限配置
├── Clipboard/
│   ├── ClipboardManager.swift  # 剪切板监控 + 历史管理
│   └── ClipboardItem.swift     # 数据模型
├── UI/
│   ├── ClipboardPanel.swift    # 底部弹出面板
│   └── ClipboardItemView.swift # 单条记录视图
├── Resources/
│   ├── Assets.xcassets         # 图片资源
│   └── AppIcon.icns            # 应用图标
└── project.yml                # XcodeGen 配置
```

### 核心设计
- **LSUIElement = true** - 隐藏 Dock 图标，作为菜单栏应用运行
- **NSPanel** - 使用非激活浮动面板，底部弹出
- **CGEvent/HotKey** - 全局快捷键监听
- **NSPasteboard** - 监控系统剪切板变化
- **UserDefaults** - 本地存储历史记录

## 🔧 设置说明

### 保留天数
可选择历史记录的保留时长：
- 7 天
- 30 天
- 90 天
- 180 天

超过保留期的记录会自动清理，同时支持按数量上限（200 条）双重清理。

### 开机自启
开启后会自动在 `~/Library/LaunchAgents/` 创建 plist，实现开机自动启动。
